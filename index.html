Stereoscopy4Browser

This simple HTML5 file uses the standard fullscreen mode of browsers after a mouse click to present several stereoscopic images in fullscreen mode. After starting the animation on your smartphone, please insert your mobile device into a VR headset to preview the 3D scenario. This repository is created for a Wikiversity Learning Resource about Stereoscopy.
